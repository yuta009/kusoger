西暦2000年。  
魔王、女神、佐藤武志の戦いの余波によって異世界は崩壊した。
その余波により、爆発的なエネルギーが現実世界に流れ込み、現実世界にも影響が出始める。

## 4. ゲームの流れ

### 4.1 タイトル画面

- タイトル表示：
- 下記ストーリーとスタートボタンを表示
	- 西暦2000年。  
	魔王、女神、佐藤武志の戦いの余波によって異世界は崩壊した。
	その余波により、爆発的なエネルギーが現実世界に流れ込み、現実世界にも影響が出始める。
	謎の生命体ガイザーが現実世界に出現し始める。

	都市の一角に設けられた小さな**避難所**。
	混乱する人々の中で、その前に立つのは、ただ一人の防衛者。 
	頼れるのは、己の身体と、進化する戦闘能力のみ。

### 4.2 プレイ画面
- 中央付近に **避難所**
- 主人公は **自由に移動可能**
- 敵は上下左右・斜めすべての方向から出現する。
- 主人公は **自動で最も近い敵を攻撃**
	- キーを押すことで特殊攻撃を発動
- 主人公の位置はA(左側)、S(下側)、D(右側)、W(上側)を移動。

### 4.3 強化選択
- 敵を一定数倒すと一時停止
- **攻撃強化を3択で選択**

### 4.4 終了
- 避難所HPが0 → ゲームオーバー
- 規定Wave生存 → クリア

## 8. 攻撃システム（自動）

### 攻撃ロジック

1. 攻撃範囲内の敵を探索
2. 最短距離の敵を選択
3. 攻撃タイプに応じて弾・エフェクト生成
4. 命中時にダメージ計算


---
## 9. 攻撃タイプ（切替・成長）

※ 初期は1種類のみ  
※ 強化選択で増加・進化  
※ レベル10到達時に「進化先」を選択し、対応する特殊攻撃を習得

### 通常攻撃（進化先により変化）

| 攻撃   | 内容                             |
| ---- | ------------------------------ |
| 通常攻撃 | 単体(斬撃を飛ばす。敵にあたったら消滅)           |
| 貫通攻撃 | 複数ヒット(斬撃を飛ばし、貫通する)             |
| 拡散弾  | 扇状に発射(斬撃が扇状に発生。一定範囲内の全ての敵にあたる) |
| レーザー | 即時ヒット(貫通かつ攻撃速度が速い)             |
| 爆発弾  | 範囲ダメージ(斬撃が当たった瞬間爆発し、周囲にダメージ)   |
|      |                                |

### 進化と特殊攻撃（レベル10で選択）

| 進化先 | 特殊攻撃 | 内容 |
| --- | --- | --- |
| 進化A（防衛型） | 防衛砲撃（Guardian Burst） | 避難所中心に全方向へ貫通弾を放つ。群れ処理向き。 |
| 進化B（制圧型） | クロノブレイク | 短時間、敵の移動停止＋プレイヤー攻撃速度UP。ボス対策向き。 |
| 進化C（機動型） | 紅蓮ダッシュ | 指定方向へ瞬間ダッシュし、軌跡に範囲ダメージ。回避・突破向き。 |

## 10. 強化システム（最重要）

### 強化発生条件

- 敵撃破数が一定数に達すると発動
- ゲームを一時停止して表示

### 強化3択（例）

- 攻撃力 +20%
- 攻撃速度 +25%
- 攻撃範囲 +15%
- 弾数 +1
- 貫通回数 +1
- 爆発範囲追加

※ 同じ強化は重複可能  
※ 後半は強力な組み合わせが生まれる

## 12. Wave構成

- Waveは時間制 or 撃破数制
    
- Wave進行で以下が増加
    - 出現数
    - 同時出現数
    - 敵HP倍率
