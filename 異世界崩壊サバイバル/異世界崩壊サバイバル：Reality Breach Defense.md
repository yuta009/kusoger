## **2. ストーリー**

### **新たなる時代の幕開け**

西暦2000年――  
魔王、女神、そして佐藤武志。  
三者の最終決戦は予想外の形で終わった。  
ぶつかり合った魔力と神力は異世界そのものを崩壊させ、  
余波としてその莫大なエネルギーが時空の裂け目から現実世界へと流れ込んだ。

その結果、現代の東京・大阪・世界各地に、  
**エルフ、ドワーフ、獣人、ドラゴン、魔物、魔王軍の残滓**が出現。  
世界は“異世界災害”という未曽有の危機にさらされる。

プレイヤーは、「現代に現れた住民たち（若者・警察・自衛隊）」の指揮官となり、  
迫り来る異世界魔物から街を守らなければならない。

---

## **3. ジャンル**

**リアルタイム拠点防衛 × 資源管理 × ミニタワーディフェンス**

- 操作はシンプル（クリック中心）
    
- リアルタイムで魔物が侵入
    
- 資源を投入して「現代兵器 × 異世界素材」のハイブリッド防衛装備を開発
    
- 10〜30代が短時間で遊べる高速テンポ

# **6. ゲームの流れ**

### **1) タイトル画面**

- ゲーム開始ボタン
    
- 世界観の短い説明
    
- 音量設定（BGM/SE）
    

### **2) メインゲーム開始**

1. マップ中央に「避難シェルター（HP 100%）」
    
2. 四方から魔物がランダムに出現
    
3. プレイヤーは資源（エネルギー）を使い防衛装置を配置
    
4. ウェーブが進むごとに敵の強化
    
5. シェルターHPが0になるとゲームオーバー
    

### **3) ゲームクリア**

- **Wave10を防衛成功**  
    → 最終リザルト画面へ
    

### **4) ゲームオーバー**

- シェルター破壊  
    → スコア送出・リトライ可
    

---

# **7. 勝敗条件**

### **勝利条件**

- **全10ウェーブを守り切る**
    
- または、**合計3分の耐久時間**をクリア（モードによる）
    

### **敗北条件**

- シェルターHP 0％
    
- 重要ラインを敵が10体突破

# **14. 技術的補足**

### ■ フレームレート

- **60fps**推奨（`requestAnimationFrame`使用）
    

### ■ 描画方式

- **Canvas（2D）推奨**
    
    - 敵・弾・タワーの描画
        
- UIはHTML DOMで構成
    

### ■ ゲームループ

`function gameLoop(timestamp) {   update();   // 敵移動、タワー攻撃、当たり判定   render();   // Canvas描画   requestAnimationFrame(gameLoop); }`

### ■ 状態管理

`const state = {   wave: 1,   energy: 50,   crystals: 0,   science: 0,   baseHP: 100,   enemies: [],   towers: [], };`

### ■ 設置処理

- タワー選択 → フィールドクリック → 設置
    
- 設置可能位置の判定（grid）
